# AGENTS.md

This file provides guidance to agents when working with code in this repository.

## Git Commit Conventions
Используй формат: PREFIX: Описание

FEATURE: - новая функциональность
BUG: - исправление бага
REFACTOR: - рефакторинг кода
CONFIG: - изменения в конфигурации
DOCS: - документация
TEST: - добавление или изменение тестов

## Workflow
При получении задачи выполняй последовательно:

1. Написание/обновление тестов - сначала пишем тесты
2. Реализация - затем реализуем саму фичу
3. Документация - обновляем Java doc, README
4. Коммит - коммит с использованием префиксов на русском языке
5. Пуш - пуш в удалённый репозиторий
