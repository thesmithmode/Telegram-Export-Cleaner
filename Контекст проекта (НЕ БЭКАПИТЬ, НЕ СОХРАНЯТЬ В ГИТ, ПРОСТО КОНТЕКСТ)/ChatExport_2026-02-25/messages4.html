Agent Second Brain - система, где ты кидаешь боту всё подряд (голосовые, скрины, идеи), а он сам решает что с этим делать. Создаёт задачи в Todoist, сохраняет мысли в Obsidian, следит за целями.

Автор собрал то, что многие пытаются костылями склеить из 5 разных сервисов. Тут всё в одном месте.

Что цепляет:

Голосовые распознаются через Deepgram (бесплатно 200 баксов на старт). Записал мысль - бот уже разобрался куда её деть.

Встроен Claude Code как AI-агент. Ты пишешь "найди ту идею про маркетинг с прошлой недели" - он находит и показывает. Или "перенеси просроченные задачи на понедельник" - делает.

Всё твоё хранится в приватном GitHub репозитории. Никаких облаков с чужими серверами. Только ты и твой VPS за 5 баксов.

Система на MCP (Model Context Protocol) от Anthropic. Это как Lego для AI-агентов - подключаешь любой сервис: Google Workspace, Notion, GitHub. Уже есть интеграция с Todoist.

Минусы:

Настройка не для новичка. 2-3 часа тишины, терминал, VPS. Автор сделал пошаговую инструкцию на 50 страниц, но всё равно - надо не бояться командной строки.

$25 в месяц ($20 Claude Pro + $5 VPS). Не копейки.

Проект open-source, можно форкнуть и допилить под себя. Автор активен, обновляет.

OpenClaw + Cloudflare Workers: На пороге ИИ-сингулярности и «бесконечной» армии агентов

Пока мы возились с деплоем на обычные VPS, подъехала тяжелая артиллерия. Cloudflare выпустили moltworker — готовую версию OpenClaw для запуска в Cloudflare Workers.

И вот почему это изменит правила игры уже с понедельника:

1. Масштабирование без границ. Cloudflare Workers — это не про один сервер. Это про глобальную инфраструктуру. Вы можете запустить сотни (или тысячи) параллельных агентов-воркеров, которые будут шуршать по своим задачам, не загружая ваш основной комп или VPS.

2. Броня от Zero Trust. Весь админ-интерфейс ассистента сразу закрывается через Cloudflare Access. Это значит, что доступ к мозгам вашего ИИ есть только у вас после SSO-авторизации (Google, GitHub и т.д.). Безопасность уровня Enterprise «из коробки».

3. Бесконечная память на R2. Все конфиги, логи и история переписок бэкапятся в Cloudflare R2 (хранилище типа S3). Контейнер может «уснуть» для экономии денег и «проснуться» с полной памятью за долю секунды.

4. Браузер на стероидах. В воркерах доступен Browser Rendering API. Агент теперь может не просто парсить текст, а полноценно рендерить страницы, делать скриншоты и видео, используя ресурсы облака, а не вашу оперативку.
Цена вопроса — $5/мес за Workers Paid план, и у вас в руках инструмент для создания целого штата автономных сотрудников, которые никогда не спят.

Если DigitalOcean — это был «комфортный переезд», то Cloudflare — это выход в открытый космос. Похоже, на следующей неделе нас ждет взрывной рост агентских сетей.

На сервере (VPS) — варианты:

1. Headless Chrome ✅

OpenClaw поддерживает headless режим (браузер без GUI):

# В конфиге OpenClaw
"browser": {
  "headless": true
}

Проблема: Как залогиниться в Claude без экрана?

Решения:

• VNC/noVNC — один раз подключиться через браузер, залогиниться, потом headless
• Перенести cookies — залогиниться локально, скопировать профиль на сервер
• Playwright login — автоматизировать логин (но нужны credentials в коде)
2. Browser Use API (облачный браузер) ✅

Есть скилл browser-use — браузер в облаке, управляется через API:

• Не нужен свой Chrome
• Работает с любого сервера
• Платный (~$0.01 за сессию)
3. Session Cookie 🔶

Можно достать sessionKey cookie из браузера и использовать curl:

• Работает ~30 дней
• Потом нужно обновлять вручную

HTTP 401 в OpenClaw — куда копать:

1. Какой именно токен?
Смотреть в логах что именно отвалилось:

openclaw logs --tail 50

Или:

cat ~/.openclaw/logs/gateway.log | grep -i "401\|token\|auth"

2. Самые частые причины:

| Сервис        | Где проверить                            |
| ------------- | ---------------------------------------- |
| Anthropic API | https://console.anthropic.com → API Keys |
| OpenAI API    | https://platform.openai.com/api-keys     |
| OpenRouter    | https://openrouter.ai/keys               |
| Telegram Bot  | @BotFather → /mybots → токен             |

3. Быстрая проверка API ключа:

# Anthropic
curl -H "x-api-key: $ANTHROPIC_API_KEY" \
  https://api.anthropic.com/v1/messages \
  -d '{"model":"claude-sonnet-4-20250514","max_tokens":10,"messages":[{"role":"user","content":"hi"}]}'

# OpenAI  
curl -H "Authorization: Bearer $OPENAI_API_KEY" \
  https://api.openai.com/v1/models

4. Если ключ в .env:

cat ~/.openclaw/.env | grep -i key

Проверить что нет лишних пробелов/кавычек.

5. Перегенерировать и обновить:

# Обновить в конфиге
nano ~/.openclaw/.env
# Рестарт
openclaw gateway restart


Если Telegram Bot Token:

• @BotFather → /revoke → получить новый
• Обновить TELEGRAM_BOT_TOKEN в .env
