**Context:** Создание и отладка skill `/wiki-init` для автономной установки claude-memory-compiler в любом проекте

**Key Exchanges:**

**Decisions Made:**
- Skill полностью автономный, без confirmation dialog (только 2 исключения: uv не установлен, paths уже tracked)

- Append-only / merge-only инварианты для `.gitignore` и `settings.local.json`
- Никаких auto-commit внутри skill

**Lessons Learned:**
- `git check-ignore` для несуществующих директорий требует trailing slash (`.memory/` не `.memory`) — баг найден при live-тестировании, пофикшен в skill
- `cd .memory && uv run python hooks/...` — кросс-платформенно работает в bash на Windows (Git Bash) и Linux
- При проверке skill dry-run в голове ("проиграй сценарий") — эффективный способ найти edge cases до боевого запуска
