# Telegram Bot

Бот реализован в `ExportBot` и работает только в личных чатах (`private`).

## Команды

- `/start` — сбросить сессию и показать справку.
- `/cancel` — отменить активный экспорт пользователя.

## Поддерживаемый ввод

- `@username`
- `https://t.me/username`

Числовые chat ID напрямую в пользовательском диалоге бот не принимает.

## Диалог экспорта

1. Пользователь отправляет `@username` или ссылку.
2. Бот показывает меню диапазона:
   - `📦 Весь чат`
   - `⏱ 24 часа`
   - `🗓 3 дня`
   - `🗓 7 дней`
   - `🗓 30 дней`
   - `📅 Указать диапазон дат`
3. Для ручного диапазона дата вводится в формате `дд.мм.гггг`.
4. После постановки задачи бот отправляет сообщение с task ID и кнопкой `❌ Отменить экспорт`.

## Ограничения и защита от дублей

- Один пользователь может иметь только один активный экспорт (`active_export:{userId}`).
- Повторный запрос при активной задаче отклоняется с подсказкой дождаться завершения или нажать `/cancel`.

## Состояния сессии

`UserSession` хранит состояние диалога и даты между сообщениями пользователя.
Старые сессии очищаются по расписанию (`evictStaleSessions`).

## Очереди и приоритет

`ExportJobProducer` отправляет задачу:
- в `telegram_export_express`, если чат вероятно уже в кэше;
- иначе в `telegram_export`.

Это ускоряет повторные экспорты по уже обработанным чатам.
